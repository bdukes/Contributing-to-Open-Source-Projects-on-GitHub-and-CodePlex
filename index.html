Collaboration – Submitting Bugs

    • Patch with failing unit test
  • Only one issue at a time

Source Control – Workflow

  1. Get latest
    • git fetch --all; git merge repo/branch
  2. Branch
    • git checkout -b DNN-12345678
  3. ????
  4. Verify change(s)
    • git status
    • git difftool
  5. Commit change(s)
    • git add -A; git commit
  6. Publish change(s)
    • git push -u origin DNN-12345678
    • Submit pull request from GitHub.com
  7. Revert to latest
    • git checkout development
